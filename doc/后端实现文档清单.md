# 后端实现文档清单

## 项目概述
- 基于Go语言和Gin框架的ERP系统后端
- 使用MySQL数据库
- 包含CRM、销售、库存、采购、财务、生产、人力资源等模块

## 核心技术栈
- Go 1.23.0
- Gin框架
- GORM ORM
- MySQL 8.0+
- JWT认证
- Viper配置管理

## 数据库结构
- 已生成完整的SQL建表语句文件：`sql/erp_database.sql`
- 包含所有模块的表结构、索引和外键约束

## 需要实现的模块

### 1. 系统模块
- **用户管理**：登录、注册、权限控制
- **角色管理**：角色创建、编辑、权限分配
- **权限管理**：权限定义、权限分配
- **日志管理**：操作日志记录、查询

### 2. CRM模块
- **客户管理**：客户信息维护、查询
- **联系人管理**：联系人信息维护、查询
- **客户活动**：活动记录、跟进
- **客户反馈**：反馈记录、处理
- **销售线索**：线索管理、转化
- **销售机会**：机会管理、跟进
- **营销活动**：活动策划、执行、效果分析

### 3. 销售模块
- **客户管理**：客户信息维护、信用评估
- **产品管理**：产品信息维护、定价
- **销售报价**：报价单创建、审批、管理
- **销售订单**：订单创建、确认、跟踪
- **销售发货**：发货单创建、确认、跟踪
- **销售发票**：发票创建、管理、收款
- **销售退货**：退货单创建、审批、处理

### 4. 库存模块
- **仓库管理**：仓库信息维护
- **库位管理**：库位信息维护
- **物料管理**：物料信息维护、分类
- **库存管理**：库存查询、库存调整
- **库存交易**：交易记录、查询
- **库存盘点**：盘点计划、执行、差异处理

### 5. 采购模块
- **供应商管理**：供应商信息维护、评估
- **采购计划**：计划创建、审批、管理
- **采购订单**：订单创建、确认、跟踪
- **采购收货**：收货单创建、确认、入库
- **采购发票**：发票管理、付款
- **采购退货**：退货单创建、审批、处理

### 6. 财务模块
- **会计科目**：科目维护、分类
- **凭证管理**：凭证创建、审核、过账
- **固定资产**：资产登记、折旧、处置
- **预算管理**：预算编制、执行、分析
- **成本中心**：成本中心维护、成本分配

### 7. 生产模块

- **主生产计划**：MPS编制、执行、跟踪

- **生产订单**：订单创建、释放、跟踪
- **生产工单**：工单创建、执行、报工
- **工艺路线**：工艺路线维护、版本管理
- **工作中心**：工作中心维护、产能管理

### 8. 人力资源模块

- **部门管理**：部门结构维护、人员分配
- **职位管理**：职位信息维护、职责定义
- **考勤管理**：考勤记录、统计、分析
